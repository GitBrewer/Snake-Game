CURSOR C_IAMARKS IS

C_B NUMBER;
C_C NUMBER;

C_AV NUMBER;

OPEN C_IAMARKS;
LOOP

5. Categorize students based on the following criterion:

If FinalCIE< 12 then CAT = ‘Weak’
If FinalCIE = 12 to 16 then CAT = ‘Average’
Give these details only for 8th semester A, B, and C section students.
